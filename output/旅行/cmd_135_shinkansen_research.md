# 鹿児島中央→倉敷 新幹線ルート・料金調査レポート

**調査日**: 2026年02月13日
**調査者**: 足軽1号

## 1. 最適ルート

### 基本ルート
**鹿児島中央 → 博多 → 岡山 → 倉敷**

```
鹿児島中央 ━━━━━━━━━━━━━> 岡山 ━━━━━> 倉敷
         九州新幹線+山陽新幹線           JR在来線
         (みずほ・さくら直通)
```

### 区間詳細

| 区間 | 路線 | 距離 | 所要時間 | 備考 |
|------|------|------|----------|------|
| 鹿児島中央→岡山 | 九州新幹線+山陽新幹線 | 730.9km | 約3〜3.5時間 | 直通列車あり |
| 岡山→倉敷 | JR山陽本線（在来線） | 約17km | 約17分 | 普通列車 |

### 直通列車について
- **みずほ**: 鹿児島中央〜新大阪間で運行。岡山駅に停車。朝夕中心に8往復。
- **さくら**: 鹿児島中央〜新大阪間で運行。岡山駅に停車（全列車停車駅）。

**重要**: どちらの列車も岡山駅まで直通運行。博多での乗り換えは不要。

## 3. 時期による料金差

### シーズン別特急料金

新幹線の指定席特急料金は、乗車日により以下の加算・減算があります：

| 期間種別 | 加算額 | 対象期間例（2026年） |
|---------|--------|---------------------|
| 最繁忙期 | +400円 | - |
| 繁忙期 | +200円 | 3月19日〜31日 |
| 通常期 | ±0円 | 4月10日〜12日（推定） |
| 閑散期 | -200円 | 4月21日〜26日 |

**注**: 自由席特急料金は、乗車日にかかわらず通常期の指定席特急料金から530円引き（シーズン加算なし）

### 注意事項

1. **2026年料金は未確定**

   - 本レポートは2025年度の料金体系をベースに推定
   - JR各社が2026年度に料金改定を行う可能性あり
   - 実際の予約時に公式サイトで最新料金を確認すること

2. **岡山⇔倉敷の在来線運賃**

   - 片道330円、往復660円を別途加算
   - 新幹線きっぷには含まれない

3. **直通列車の確認**

   - みずほ・さくらは岡山まで直通
   - 新倉敷駅に停車する列車もあるが、倉敷駅には新幹線は停車しない
   - 岡山駅で在来線に乗り換えが必要

4. **早割の購入期限**

   - スーパー早特きっぷ: 乗車日の14日前まで
   - EX早特7: 乗車日の7日前まで
   - 購入期限を過ぎると通常料金

5. **繁忙期の座席確保**

   - 3月19〜22日は春休み期間で混雑が予想される
   - 早めの予約を推奨
