# multi-agent-shogun 操作説明書

## 起動操作

### 1. 出陣（エージェント起動）

```bash
cd ~/cc_project/multi-agent-shogun
./shutsujin_departure.sh
```

### 2. 将軍に接続（別ターミナルで）

```bash
source ~/.bashrc    # 初回のみ
css
```

または直接：

```bash
tmux attach-session -t shogun
```

### 3. 命令を出す

将軍に直接話しかける：

```
〇〇を調査せよ
```

### 4. 進捗確認（任意）

- `dashboard.md` をエディタで開く
- または `csm` で家老＋足軽の9ペインを表示

---

## 作業対象プロジェクトの指定方法

### 方法1: config/projects.yaml を編集

```yaml
projects:
  - id: my_app
    name: "My Application"
    path: "/home/saneaki/cc_project/my-app"
    priority: high
    status: active

current_project: my_app
```

### 方法2: 将軍への命令でパスを指定

```
/home/saneaki/cc_project/my-app のREADMEを更新せよ
```

## 構造

```
multi-agent-shogun/     <- ここで起動・実行（司令塔）
├── shutsujin_departure.sh
├── config/projects.yaml  <- 作業対象プロジェクトを登録
├── dashboard.md          <- 進捗確認
└── queue/                <- タスク管理

~/cc_project/my-app/    <- 作業対象（別の場所でOK）
```

---

## 終了・再接続

- **終了**: ターミナルを閉じるだけ（tmuxセッションは生き残る）
- **再接続**: `css`（将軍）/ `csm`（家老＋足軽）
