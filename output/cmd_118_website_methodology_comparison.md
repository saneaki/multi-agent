# 弁護士事務所HP作成 方法論比較レポート（統合版）

**作成日**: 2026-02-11
**コマンド**: cmd_118
**入力レポート**: ノーコード系（足軽1号）、Claude Code活用系（足軽2号）、伝統的CMS/SSG系（足軽1号）

---

## 1. エグゼクティブサマリー

倉敷市の弁護士事務所HPを自力構築するための7つの方法論を調査・比較した。結論として、**Astro + Tailwind CSS + GitHub Pages（Claude Code活用）** を第1推奨とする。

**理由**: SEO最強（静的HTML＋構造化データ）、年間コスト最安クラス（約4.7万円、Claude Code契約含む）、セキュリティリスク最小、かつClaude Codeとの親和性が最も高く、Web開発未経験でも構築・保守が可能。

ただし、Notionでのブログ更新を最優先する場合は **Next.js + Vercel（Notion API連携）** が、技術的な学習コストを最小化しデザインテンプレートを活用したい場合は **Wix ビジネスプラン** が有力な代替案となる。

## 5. 各方法論の詳細

### 5-1. Notion Sites / Super.so（ノーコード）

**概要**: Notionで作成したページをそのままWebサイトとして公開する方法。Super.soは外部サービスでNotionページにカスタムドメイン・デザインカスタマイズを追加する。

| 項目 | 内容 |
|------|------|
| 初期費用 | 約2,500〜5,000円（ドメイン代） |
| 月額 | 約1,000〜1,400円（Super.so年払い時） |
| 技術難易度 | ⭐⭐☆☆☆（容易） |
| 構築期間 | 3〜5日 |

**メリット**:
- Notionで編集するだけで即座にサイト反映（更新容易性No.1）
- 最低コスト（ノーコード系）
- メンテナンスフリー（ホスティング・SSL・セキュリティはSuper.so/Notion側が自動対応）

**デメリット**:
- デザインの自由度が低く、法律事務所に求められる「重厚感」を表現しづらい
- SEO最適化に限界（URL構造のカスタマイズ制約、サイトマップの細かい制御不可）
- お問い合わせフォームはGoogleフォーム埋め込み等に制約
- プラットフォーム依存リスク（Super.soサービス終了時の移行リスク）

### 5-2. Wix（ノーコード）

**概要**: 世界最大級のノーコードWebサイトビルダー。ドラッグ&ドロップで構築し、法律事務所向けテンプレートが豊富。

| 項目 | 内容 |
|------|------|
| 初期費用 | 約1,200〜6,800円 |
| 月額 | 約2,700円（ビジネスプラン、年払い時） |
| 技術難易度 | ⭐⭐⭐☆☆（普通） |
| 構築期間 | 1〜3週間 |

**メリット**:
- ローカルSEO最強クラス（Googleビジネスプロフィール自動連携、構造化データ自動生成）
- 法律事務所向けテンプレートが豊富で信頼感あるデザインを実現可能
- お問い合わせフォーム・予約カレンダー・チャットボットがビルトイン
- 日本語サポート充実

**デメリット**:
- 月額コストがやや高い（年間約32,400円）
- プラットフォームロックイン（他サービスへの移行が困難）
- Claude Codeとの連携不可（コード生成の恩恵を受けられない）

### 5-3. Squarespace（ノーコード）

**概要**: デザイン品質で業界トップのノーコードビルダー。ブランディング重視の事務所に最適。

| 項目 | 内容 |
|------|------|
| 初期費用 | 約2,400〜4,200円 |
| 月額 | 約3,450円（Businessプラン、年払い時） |
| 技術難易度 | ⭐⭐⭐☆☆（普通） |
| 構築期間 | 1〜3週間 |

**メリット**:
- デザインの完成度が最高（180以上のプロフェッショナルテンプレート）
- SEO基本機能完備（メタタグ、OGP、サイトマップ自動生成）
- 落ち着いたトーンのテンプレートが豊富で士業サイトに適合

**デメリット**:
- コスト最高（ノーコード系で年間約41,400円）
- ローカルSEO自動連携がWixに劣る（Googleビジネスプロフィール連携は手動）
- エディタの学習コストがやや高い
- Claude Codeとの連携不可

### 5-4. Astro + Tailwind CSS + GitHub Pages（Claude Code活用）

**概要**: 最新の静的サイトジェネレーターAstroとTailwind CSSを使い、Claude Codeでコード自動生成。GitHub Pagesで無料ホスティング。

| 項目 | 内容 |
|------|------|
| 初期費用 | 約1,500円（ドメイン代のみ） |
| 月額 | 約3,000円（Claude Code Pro。既契約なら追加0円） |
| 技術難易度 | ⭐⭐☆☆☆（Claude Codeが大半を自動生成） |
| 構築期間 | 1〜2日（10〜15時間） |

**メリット**:
- **SEO最強**: 完全な静的HTMLでクローラーが即座にDOM読み取り。Lighthouseスコア95点以上
- **ローカルSEO最適**: schema.orgのLocalBusiness/Attorneyマークアップ、構造化データをClaude Codeが自動設定
- **コスト最安クラス**: ホスティング無料、Claude Code既契約ならドメイン代のみ
- **セキュリティ最強**: 静的サイトのため攻撃対象が最小。WordPressの脆弱性リスクなし
- **Claude Code親和性最強**: プロンプト入力だけでサイト全体を5〜10分で自動生成
- **高速表示**: JavaScriptを最小限に抑えた静的HTMLでCore Web Vitals優秀
- **保守が楽**: ファイル構造がシンプル、アップデート作業不要

**デメリット**:
- ブログ更新にMarkdownファイル作成＋git push が必要（慣れれば10分）
- お問い合わせフォームは外部サービス（Googleフォーム、Netlify Forms）に依存
- Git操作の最低限の理解が必要（ただしClaude Codeがサポート）

**Claude Codeでの構築フロー（実績ベース）**:
1. プロンプトでサイト要件を入力（5分）
2. Claude Codeが全ページ・コンポーネント・SEO設定を自動生成（5〜10分）
3. コンテンツ（事務所情報・弁護士プロフィール等）を投入（3〜5時間）
4. GitHub Pagesにデプロイ＋独自ドメイン設定（1時間）

### 5-5. Next.js + Vercel（Claude Code活用）

**概要**: ReactベースのフルスタックフレームワークNext.jsを使用。Notion APIと連携し、Notionでブログを書くだけで自動的にサイトに反映。

| 項目 | 内容 |
|------|------|
| 初期費用 | 約1,500円（ドメイン代のみ） |
| 月額 | 約3,000円（Claude Code Pro。既契約なら追加0円） |
| 技術難易度 | ⭐⭐⭐☆☆（Astroよりやや複雑） |
| 構築期間 | 2〜3日（15〜20時間） |

**メリット**:
- **コンテンツ更新が最も楽**: Notionで記事を書いて「公開」にするだけで自動反映（Git操作不要）
- **拡張性最強**: 将来的に顧客ポータル、予約システム、会員制ページ等の追加が容易
- **SEO良好**: SSG/ISR（Incremental Static Regeneration）でAstroに近い速度を実現
- **Claude Code親和性良好**: プロジェクト全体の自動生成・保守が可能

**デメリット**:
- Astroより初期学習コストが高い（Reactの基本概念理解が必要）
- JavaScriptバンドルが大きく、Astroより初期ロードがわずかに遅い
- Notion API設定にやや手間（APIキー取得、環境変数管理）
- シンプルなコーポレートサイトにはオーバースペック

### 5-6. WordPress + レンタルサーバー（伝統的CMS）

**概要**: 世界シェアNo.1のCMS。法律事務所向け専用テーマが豊富で、ブラウザ上でコンテンツ管理が完結。

| 項目 | 内容 |
|------|------|
| 初期費用 | 0〜15,000円（テーマ代による） |
| 月額 | 約643〜990円（ConoHa WING/Xserver） |
| 技術難易度 | ⭐⭐☆☆☆（GUI操作で完結） |
| 構築期間 | 1〜3週間 |

**メリット**:
- **テーマ・プラグインが圧倒的に豊富**: 法律事務所専用テーマ（LAW: 11,980円、JUSTICE: 14,800円等）
- **コンテンツ更新が非常に容易**: ブラウザ上でワープロ感覚。スマホからも投稿可能
- **SEO対応力が高い**: Yoast SEO等の強力なSEOプラグインで細かい最適化可能
- **情報が豊富**: 日本語の解説記事・動画・書籍が大量に存在
- **複数人運用が容易**: 権限管理機能で事務スタッフにも運用を委任可能

**デメリット**:
- **セキュリティリスクが最大**: プラグイン脆弱性（全脆弱性の93%がプラグイン由来）、定期的なアップデート必須
- **保守作業が継続的に必要**: WordPress本体・プラグイン・テーマの更新を怠ると攻撃対象に
- **表示速度がやや遅い**: データベースアクセスが発生（静的サイトに劣る）
- **Claude Codeとの親和性が低い**: PHPテーマカスタマイズにはPHP知識が必要

**推奨サーバー**:
- **コスト重視**: ConoHa WING ベーシック（月額643円〜）
- **安定性重視**: Xserver スタンダード（月額990円〜）

**推奨テーマ**:
- **コスパ重視**: LAW (TCD031) — 11,980円、法律事務所専用
- **カスタマイズ重視**: JUSTICE — 14,800円、Elementor対応

## 6. 法律事務所HPとしての評価軸分析

### 6-1. 信頼感・専門性の訴求

法律事務所のHPは「信頼感」と「専門性」が最重要。クライアントは人生の重大な問題（離婚、相続、刑事事件等）を相談するため、サイトの第一印象が依頼の決め手となる。

| 方法論 | 信頼感の演出力 | 根拠 |
|--------|:-:|------|
| Notion/Super.so | ⭐⭐☆ | モダンでカジュアル寄り。「素人感」が出るリスク |
| Wix | ⭐⭐⭐⭐ | 法律事務所向けテンプレート豊富。落ち着いたトーン（紺・ダークグレー）のデザイン |
| Squarespace | ⭐⭐⭐⭐⭐ | デザイン品質最高。ブランディング力No.1 |
| **Astro + Tailwind** | **⭐⭐⭐⭐** | Claude Codeでプロフェッショナルなデザインを自動生成。高速表示が信頼感に直結 |
| Next.js + Vercel | ⭐⭐⭐⭐ | モダンUIで信頼感演出。カスタマイズ自在 |
| WordPress | ⭐⭐⭐⭐ | LAW/JUSTICEテーマで重厚感。実績豊富 |
| Hugo/Jekyll | ⭐⭐☆ | テーマ選択肢が少なく、カスタマイズに技術力が必要 |

### 6-2. 地域SEO（「倉敷市 弁護士」等）

弁護士サイトは「YMYL（Your Money or Your Life）」カテゴリに分類され、Googleの品質評価が厳格。地域密着型のローカルSEO対策が集客の生命線。

| 方法論 | ローカルSEO | 根拠 |
|--------|:-:|------|
| Notion/Super.so | ⭐⭐⭐ | 基本的なSEO機能はあるが、構造化データのカスタマイズに制約 |
| Wix | ⭐⭐⭐⭐⭐ | Googleビジネスプロフィール自動連携。ローカルSEO特化機能が最も充実 |
| Squarespace | ⭐⭐⭐⭐ | SEO基本機能完備。ローカルSEO連携はWixに一歩劣る |
| **Astro + Tailwind** | **⭐⭐⭐⭐⭐** | 完全な静的HTMLでインデックス速度最速。schema.org構造化データをClaude Codeが完璧に設定 |
| Next.js + Vercel | ⭐⭐⭐⭐ | SSGならAstro同等。Metadata APIで強力なSEO管理 |
| WordPress | ⭐⭐⭐⭐ | Yoast SEO等のプラグインで細かい最適化可能。ページ速度がやや弱点 |
| Hugo/Jekyll | ⭐⭐⭐ | 技術力があれば高いSEO効果だが、設定の難易度が高い |

**ローカルSEO戦略のポイント**:
- Googleビジネスプロフィール（旧Googleマイビジネス）の登録・最適化は**どの方法論でも必須**
- 構造化データ（schema.org: LocalBusiness, Attorney）でGoogle検索結果に事務所情報を表示
- 「倉敷市 弁護士 〇〇（専門分野）」等のロングテールキーワードでのコンテンツ戦略
- ブログ記事による継続的なコンテンツ発信が検索順位維持に重要

### 6-3. 問い合わせ導線の実装しやすさ

法律事務所HPの最終目標は「問い合わせ・相談予約の獲得」。各方法論でのCTA（Call to Action）実装のしやすさを評価。

| 方法論 | 問い合わせ導線 | 実装方法 |
|--------|:-:|------|
| Notion/Super.so | ⭐⭐⭐ | Notionフォーム or Googleフォーム埋め込み。デザイン統一感に欠ける |
| Wix | ⭐⭐⭐⭐⭐ | ビルトインフォーム、予約カレンダー、チャットボット、電話ボタン。全ページCTA配置が容易 |
| Squarespace | ⭐⭐⭐⭐ | ビルトインフォーム、予約機能。Wixほど多機能ではないがシンプルで使いやすい |
| Astro + Tailwind | ⭐⭐⭐⭐ | Googleフォーム埋め込み or Netlify Forms。電話番号・メールリンクはClaude Codeが自動実装 |
| Next.js + Vercel | ⭐⭐⭐⭐⭐ | カスタムフォーム（Supabase保存）、メール通知、予約システムも自力実装可能 |
| WordPress | ⭐⭐⭐⭐⭐ | Contact Form 7、WPForms等のプラグインで多機能フォーム。予約プラグインも豊富 |
| Hugo/Jekyll | ⭐⭐ | 外部サービス（Googleフォーム、Calendly等）に完全依存 |

## 7. 推奨案

### 第1推奨: Astro + Tailwind CSS + GitHub Pages（Claude Code活用）

**推奨理由**:

1. **SEO最強 × コスト最安**: 静的HTMLでLighthouseスコア95点以上を実現しつつ、ホスティング無料。Claude Code既契約ならドメイン代（年間約1,500円）のみの追加コスト
2. **Claude Codeとの親和性が最高**: プロンプト1つでサイト全体を自動生成。デザイン変更もClaude Codeへの指示だけで完了。Web開発未経験でも構築可能
3. **セキュリティリスク最小**: 静的サイトのためWordPressのような脆弱性リスクがない。法律事務所として個人情報保護の観点で安心
4. **保守負担ゼロ**: サーバー管理・データベース管理・セキュリティアップデートが不要
5. **地域SEO最適**: schema.org構造化データ（LocalBusiness, Attorney）をClaude Codeが完璧に自動設定

**注意点**:
- ブログ更新にはMarkdown + git push が必要（慣れれば10分/回）
- お問い合わせフォームは外部サービス利用（Googleフォーム or Netlify Forms）

**実行ロードマップ**:

| フェーズ | 内容 | 期間 |
|---------|------|------|
| Phase 1 | 環境準備（Node.js, GitHubリポジトリ作成） | 1時間 |
| Phase 2 | Claude Codeでサイト自動生成（全ページ・SEO・構造化データ） | 2〜3時間 |
| Phase 3 | コンテンツ投入（事務所情報、弁護士プロフィール、業務内容） | 3〜5時間 |
| Phase 4 | GitHub Pagesデプロイ + 独自ドメイン設定 | 1時間 |
| Phase 5 | Googleビジネスプロフィール登録 + Search Console連携 | 1〜2時間 |
| **合計** | | **1〜2日（約10〜15時間）** |

**年間コスト概算**: 約37,500円（Claude Code Pro月額3,000円 × 12 + ドメイン1,500円）
※Claude Codeを他用途でも使用している場合: **約1,500円/年（ドメイン代のみ）**

### 第2推奨: Next.js + Vercel + Notion API（Claude Code活用）

**推奨理由**:

1. **コンテンツ更新が最も楽**: Notionで記事を書いて「公開」にするだけ。Git操作不要
2. **将来の拡張性**: 顧客ポータル、予約システム等の高度な機能を後から追加可能
3. **Notion資産の活用**: 既存のNotionワークスペースとシームレスに連携

**注意点**:
- Astroよりやや複雑（Reactの基本概念の理解が推奨）
- Notion API設定に初期の手間あり

**実行ロードマップ**:

| Phase 2 | Claude Codeでサイト自動生成 + Notion API連携 | 3〜5時間 |

**年間コスト概算**: 約37,500円（Claude Code Pro月額3,000円 × 12 + ドメイン1,500円）

### 第3推奨: Wix ビジネスプラン

**推奨理由**:

1. **Claude Codeを使わない選択肢として最適**: 技術的な学習コストが最小
2. **ローカルSEO機能が充実**: Googleビジネスプロフィール自動連携が強力
3. **法律事務所テンプレートが豊富**: テキスト・画像を差し替えるだけで信頼感あるサイトが完成
4. **オールインワン**: フォーム、予約、チャットボットがビルトイン

**注意点**:
- 年間コストが約32,400円でAstroの約2倍（Claude Code不使用の場合はAstroのドメイン代のみ1,500円と大差）
- プラットフォームロックイン（移行困難）
- Claude Codeのスキルを活用できない

**実行ロードマップ**:

## 8. まとめ

### 推奨の判断フロー

```
Q1: Claude Codeを日常的に使っていますか？
├── はい → Q2へ
└── いいえ → Wix ビジネスプラン（第3推奨）

Q2: ブログをNotionで頻繁に更新しますか（週3回以上）？
├── はい → Next.js + Vercel + Notion API（第2推奨）
└── いいえ → Astro + Tailwind + GitHub Pages（第1推奨）
```

### 方法論別の適合ユーザー

| 推奨順位 | 方法論 | 最適なユーザー像 |
|---------|--------|-----------------|
| **第1推奨** | Astro + Tailwind + GitHub Pages | Claude Code使用者。SEO・コスト・セキュリティを最優先。ブログ更新は週1回程度 |
| **第2推奨** | Next.js + Vercel | Claude Code使用者。Notionでの頻繁なブログ更新を重視。将来の機能拡張も視野 |
| **第3推奨** | Wix ビジネスプラン | Claude Code非使用者。技術学習なしでプロ品質のサイトを構築したい |
| 代替案A | WordPress + ConoHa WING | ブログ更新を最重視。テーマ・プラグインの豊富さと日本語情報を活用したい |
| 代替案B | Notion Sites / Super.so | 最低コスト重視。サイトは名刺代わりで集客はGoogleビジネスプロフィール主体 |

### 最終結論

殿のスキルセット（Claude Code・GitHub・Notionを日常使用）を考慮すると、**Astro + Tailwind CSS + GitHub Pages（Claude Code活用）** が最も合理的な選択である。

- **構築**: Claude Codeへのプロンプト1つで1〜2日で完成
- **コスト**: 年間約1,500円（ドメイン代のみ。Claude Code既契約前提）
- **SEO**: 静的HTML＋構造化データで「倉敷市 弁護士」上位表示に最適
- **セキュリティ**: 攻撃対象が最小。法律事務所として安心
- **保守**: Claude Codeに指示するだけでデザイン変更・コンテンツ更新が可能

ブログ更新の手軽さを最優先する場合のみ、Next.js + Vercel（Notion API連携）への切り替えを検討されたい。
