## 4. 諸費用

### 4.1 物件購入時の諸費用（物件価格の5～10%）

| 費用項目 | 費用相場 |
|---------|---------|
| 仲介手数料 | 物件価格の3%+6万円+消費税（上限） |
| 不動産取得税 | 固定資産税評価額×3%（軽減措置後） |
| 登録免許税（所有権移転） | 固定資産税評価額×1.5%（2026年3月31日まで） |
| 司法書士報酬 | 5～10万円 |
| 印紙税（1,000～5,000万円の売買契約） | 10,000円（軽減措置後、2026年3月31日まで） |
| 固定資産税精算金 | 数万円 |

**例**: 2,500万円の物件の場合
仲介手数料: (2,500万円×3% + 6万円)×1.1 = 90.2万円
諸費用合計（概算）: 125～250万円（物件価格の5～10%）

## 7. 推奨パターンと考察

### 7.1 推奨パターン: **パターンB（築15年 + 中リフォーム）**

**理由**:
1. **総費用が最も低い（3,685万円）**: 初期投資を抑えられる
2. **物件価格とリフォーム費用のバランスが良い**: 2,500万円の物件に820万円のリフォームで希望仕様を実現
3. **築15年は適度な状態**: 築浅ほど高額ではなく、築古ほど劣化していない
4. **リフォーム自由度が高い**: 構造的な問題が少なく、希望通りの間取り変更・設備導入が可能
5. **補助金活用の必要なし**: 耐震補強が不要（新耐震基準）で、手続きが簡素

### 7.2 代替案: **パターンD（旧耐震 + フルリノベ + 補助金活用）**

**理由**:
1. **物件価格が最も低い（1,200万円）**: キャッシュフローに余裕が生まれる
2. **倉敷市の耐震補助金を活用**: 耐震補強費150万円のうち115万円が補助、実質35万円の負担
3. **全面リノベで長期使用可能**: 配管・配線を全面更新し、今後20年以上使用可能
4. **希望仕様を完全実現**: フルリノベーションで弁護士事務所に最適化
5. **総費用は3,742万円**: パターンBより57万円高いが、長期的なコストパフォーマンスは良好

**注意点**:
- 工期が長い（6ヶ月～1年）
- 旧耐震のため、将来の売却時に不利になる可能性
- 築35年のため、15～20年後に再度大規模修繕が必要になる可能性
