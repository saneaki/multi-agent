## 1. 日本政策金融公庫の創業融資

### 新規開業・スタートアップ支援資金（旧：新創業融資制度）

2024年4月より「新創業融資制度」が廃止され、「**新規開業・スタートアップ支援資金**」が一部拡充されて後継制度となりました。

| 項目 | 内容 |
|------|------|
| **融資限度額** | 最大7,200万円（うち運転資金4,800万円） |
| **返済期間** | 設備資金：20年以内<br>運転資金：10年以内 |
| **据置期間** | 5年以内 |
| **金利** | 日本政策金融公庫の基準利率を適用（資金使途・担保の有無・条件により変動）<br>※弁護士・士業向けの特別金利設定は公開情報なし |
| **担保・保証人** | 無担保・無保証人でも申請可能（審査により判断） |
| **自己資金比率** | 創業資金総額の1/10以上が目安（審査で重視） |

### 必要書類

- 創業計画書
- 見積書（設備資金の場合）
- 事業計画書（収支見込み）
- 自己資金を証明できる書類（通帳コピー等）

### 審査のポイント

1. **事業計画の妥当性**: 倉敷市の法律サービス需要、競合分析、差別化戦略
2. **自己資金比率**: 総投資額の10%以上が望ましい（例：4,000万円の場合400万円）
3. **返済能力**: 月次収支計画、依頼者数の見込み、着手金・報酬体系の明確化

## 3. 住宅ローンの適用可否（自宅兼事務所）

### 適用条件

| 項目 | 条件 |
|------|------|
| **床面積比率** | 居住部分が全体の50%以上であること |
| **利用目的** | 事前に金融機関に説明し、承認を得ること |
| **事業部分の用途** | 申込本人または同居者が生計を営むための自己使用（賃貸不可） |
| **区画要件（フラット35）** | 住宅部分と事業部分が壁・建具で区画され、相互に行き来可能であること |

### 住宅ローン控除の適用

| 事業部分の割合 | 住宅ローン控除の適用 |
|----------------|----------------------|
| 10%未満 | 住宅ローン控除を100%受けられる |
| 10%〜50%未満 | 居住部分の割合に応じて控除可能<br>（例：居住60%、事業40% → 60%が控除対象） |
| 50%以上 | 住宅ローン控除の適用なし |

### 留意点

- **最初から自宅兼事務所として計画する場合**: 建築計画を金融機関に事前説明し、物件全体を住宅ローンの対象とすることが可能
- **住宅ローン控除と必要経費の併用**: 居住部分は控除、事業部分は必要経費として処理可能

## 5. ランニングコスト比較（新築 vs 中古リフォーム）

### 年間ランニングコスト比較表（目安）

| 項目 | 新築（40坪） | 中古リフォーム（築20年、40坪） | 差額 |
|------|-------------|--------------------------------|------|
| **固定資産税** | 15万円<br>（3年間は1/2減額措置） | 10万円<br>（建物評価額低） | △5万円 |
| **都市計画税** | 3万円 | 2万円 | △1万円 |
| **火災保険** | 4万円/年<br>（10年一括：40万円） | 5万円/年<br>（木造・築古は割高） | +1万円 |
| **地震保険** | 2万円/年 | 3万円/年 | +1万円 |
| **修繕・メンテナンス** | 5万円/年<br>（設備保証あり、当初10年は少） | 15万円/年<br>（配管・外壁等の経年劣化） | +10万円 |
| **光熱費** | 18万円/年<br>（省エネ設備・高断熱） | 24万円/年<br>（断熱性能低、旧式設備） | +6万円 |
| **設備更新費（積立目安）** | 5万円/年<br>（10〜20年後の設備交換） | 10万円/年<br>（早期交換必要） | +5万円 |
| **合計** | **52万円/年** | **69万円/年** | **+17万円/年** |

※ 固定資産税は新築後4年目以降は年間30万円程度に増加する可能性あり

### 固定資産税の減額措置

- **新築住宅**: 3年間（認定長期優良住宅は5年間）、建物の税額が1/2に減額
- **中古住宅**: 減額措置なし。ただし築20年以上は評価額の下限に近く、リノベーションしても新築ほど高くならない

### 中長期的な視点

- **新築のメリット**: 当初10年は修繕費が少なく、設備保証により安心。光熱費も削減可能
- **中古リフォームのリスク**: 躯体の劣化（配管・基礎）、予期せぬ修繕費、10〜15年後に建て替え・住み替えが必要になる可能性

### 6-3. 防犯・セキュリティ要件

| 設備 | 推奨仕様 |
|------|----------|

- **事務所入口**: 内部スタッフの入退室記録
- **画像伝送機能**: 外部サーバーへの記録保存（セコム・ALSOK等の機械警備との連動を推奨）

#### （3）セキュリティゾーニング

2. **相談室（半専用ゾーン）**: 予約顧客のみアクセス
3. **事務所エリア（専用ゾーン）**: カードリーダー等で入退室管理、外部者は立入禁止

#### （3）障害者差別解消法に基づく合理的配慮（バリアフリー）

**2024年4月1日より義務化**: 民間事業者による合理的配慮の提供が法的義務に変更

**具体的な配慮の例**:
- **物理的障壁の除去**: スロープ設置、段差解消、自動ドア
- **車椅子利用者への補助**: 携帯スロープの設置、キャスター上げ
- **多機能トイレ**: 設置推奨（義務ではないが、合理的配慮の一環）
- **駐車場**: 車椅子使用者用駐車スペースの確保（公共施設は義務、民間は推奨）

**補助制度**: 多くの地方公共団体がバリアフリー工事施工費を補助（倉敷市に確認）

## 8. まとめ

### 推奨する資金調達の組み合わせ例

**ケース1: 新築・土地購入（総投資額5,000万円）**
- 自己資金: 500万円（10%）
- 日本政策金融公庫（新規開業資金）: 2,000万円（設備資金・運転資金）
- 中国銀行（事業用不動産ローン）: 2,500万円（土地・建物）
- 岡山県信用保証協会保証: 併用

**ケース2: 中古リフォーム（総投資額3,500万円）**
- 自己資金: 350万円（10%）
- 日本政策金融公庫（新規開業資金）: 1,500万円
- 民間銀行（事業用ローン）: 1,650万円
- 補助金（IT導入・小規模事業者持続化）: 100万円〜300万円

### ランニングコストの長期的視点

- **新築**: 初期投資は高いが、10年間の修繕費・光熱費が抑えられる（年間17万円削減）
- **中古リフォーム**: 初期投資は安いが、築20年以降の修繕費増大リスクあり（10〜15年後に建て替え検討の可能性）
