## 1. 面積・間取り要件定義

### 1.1 各部屋の標準面積

弁護士事務所は、機密性の高い個室での打合せが必要であり、保管すべき書類や参考図書が通常のオフィスよりも多いため、一人当たりの面積が**4坪以上**と大きくなる傾向がある。業界では「**弁護士の人数×5坪+10坪**」が一つの目安とされる。

| 部屋 | 用途・仕様 | 最小面積 | 推奨面積 | 余裕あり |
|------|-----------|---------|---------|---------|
| **相談室** | 防音仕様、依頼者2-3名+弁護士1名 | 10㎡ (3坪) | 13㎡ (4坪) | 16㎡ (5坪) |
| **弁護士執務室** | 2名分 (個室または共有)、書棚含む | 26㎡ (8坪) | 33㎡ (10坪) | 40㎡ (12坪) |
| **事務員スペース** | 2名分、デスク・キャビネット・来客対応カウンター | 20㎡ (6坪) | 26㎡ (8坪) | 33㎡ (10坪) |
| **待合室** | 来客3-4名が待機できるスペース | 10㎡ (3坪) | 13㎡ (4坪) | 16㎡ (5坪) |
| **書庫/資料室** | 法律書、事件ファイル保管 | 10㎡ (3坪) | 13㎡ (4坪) | 16㎡ (5坪) |
| **トイレ** | 来客用・職員用（バリアフリー対応） | 6㎡ (2坪) | 10㎡ (3坪) | 13㎡ (4坪) |
| **給湯室/ミニキッチン** | 簡易調理設備 | 3㎡ (1坪) | 6㎡ (2坪) | 10㎡ (3坪) |
| **廊下・共用部** | 動線確保 | 10㎡ (3坪) | 13㎡ (4坪) | 16㎡ (5坪) |

### 1.2 合計必要面積（3パターン）

| パターン | 合計延床面積 | 坪換算 | 備考 |
|---------|------------|--------|------|
| **最小** | 95㎡ | 29坪 | ギリギリの運用。将来の拡張余地なし |
| **推奨** | 127㎡ | 39坪 | 快適な業務環境。標準的な配置 |
| **余裕あり** | 160㎡ | 49坪 | 将来の人員増加・個室増設に対応可能 |

**業界目安との比較**:
- 弁護士2名×5坪+10坪 = **20坪** (最小限の目安)
- 本調査の「推奨」39坪は、事務員スペース・来客対応設備を考慮した実務的な水準

### 1.3 平屋 vs 2階建ての間取り例

#### 平屋 (推奨127㎡の場合)

**メリット**:
- バリアフリー対応が容易
- 動線がシンプルで来客案内がスムーズ
- 構造がシンプルで建築コスト抑制

**デメリット**:
- 広い敷地が必要 (建ぺい率60%の場合、約211㎡ = 64坪の敷地が必要)

**間取りイメージ**:
```
┌────────────────────────────────┐
│ 待合室 │ 相談室(防音) │ トイレ │
├────────────────────────────────┤
│ 事務員スペース │ 書庫 │ 給湯室 │
├────────────────────────────────┤
│ 弁護士執務室（2名） │ 資料室 │
└────────────────────────────────┘
```

#### 2階建て (推奨127㎡の場合)

**メリット**:
- 敷地面積を抑えられる (建ぺい率60%の場合、約106㎡ = 32坪の敷地)
- 1階を来客対応、2階を執務エリアに分離可能 (プライバシー確保)

**デメリット**:
- 階段スペースが必要 (約10㎡追加)
- バリアフリー対応にエレベーター設置が必要な場合あり

**間取りイメージ**:
```
【1階】
┌────────────────────────────────┐
│ 待合室 │ 相談室(防音) │ トイレ │
├────────────────────────────────┤
│ 事務員スペース │ 書庫 │ 給湯室 │
└────────────────────────────────┘

【2階】
┌────────────────────────────────┐
│ 弁護士執務室（2名） │ 資料室 │
└────────────────────────────────┘
```

### 3.2 諸費用（土地購入関連）

土地購入にかかる諸費用は、一般的に**土地価格の5-10%**とされる。

| 項目 | 計算式 | 推奨パターン (2階建て、土地1,925万円) の金額 |
|------|--------|-----------------------------------|
| **仲介手数料** | (土地価格 × 3% + 6万円) × 1.1 (消費税) | 約69.5万円 |
| **測量費** | 土地家屋調査士費用 (境界確定測量) | 40-80万円 (約60万円) |
| **不動産取得税** | 土地価格 × 3% (住宅用地の場合軽減措置あり) | 約58万円 (軽減前) |
| **登録免許税** | 土地価格 × 2% | 約38.5万円 |
| **印紙税** | 売買契約書 (1千万円超5千万円以下) | 2万円 |
| **司法書士報酬** | 所有権移転登記手続き | 10-15万円 (約12万円) |
| **その他** | 行政書類取得費等 | 5万円 |

**諸費用合計 (推奨パターン)**: 約245万円 (土地価格の約12.7%)

## 4. 結論と推奨

### 4.1 推奨プラン

**建物**: 2階建て、延床面積127㎡ (39坪)
**土地**: 55坪 (182㎡)、倉敷駅徒歩13-15分、第一種中高層住居専用地域
**総費用** (土地のみ): 約2,285万円

**理由**:
1. **コスト効率**: 平屋より土地面積を抑えられ、土地代を約1,000万円削減可能
2. **業務効率**: 1階を来客対応、2階を執務エリアに分離でき、プライバシー確保と動線最適化が両立
3. **立地適正**: 駅徒歩13-15分は弁護士事務所として十分なアクセス性を保ちつつ、坪単価を抑制
4. **用途地域**: 第一種中高層住居専用地域は法律事務所の立地として問題なく、静かな環境

### 4.2 次のステップ

1. **建築コストの算定** → 別途調査 (subtask_114b 等で対応)
2. **融資・資金調達計画** → 別途調査 (subtask_114c 等で対応)
3. **具体的な物件候補の絞り込み** → 不動産会社との連携
4. **用途地域・建築基準法の詳細確認** → 倉敷市建築指導課への問い合わせ
